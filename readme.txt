This is a project for my current masters in computer science at Memorial University of Newfoundland, St. Johns, Canada.

This is a tool written to evaluate the performance of privacy preserving techniques in big data.

To Run/Execute
Step 1: Install TCL 
Step 2: Download the project
Step 3: Browse to the folder containing the file tool.tcl
Step 4: At shell command prompt type the following command
	tclsh tool.tcl &
or
	wish tool.tcl &
